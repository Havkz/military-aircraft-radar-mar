# Security Policy

## Supported version

Security fixes are applied to the latest version on the `main` branch.

## Reporting a vulnerability

Please use GitHub's **Report a vulnerability** form in the repository's Security tab. Do not disclose exploitable details, private locations, credentials, or personal data in a public issue.

Include the affected version, Android version, reproduction steps, impact, and any suggested mitigation. Reports will be assessed as time permits; this community project does not guarantee a specific response or resolution time.

General bugs that do not have a security impact should use the public bug-report template.
